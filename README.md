# How to

## Build & run the backend

- `mvn clean package`
- `java -jar target/chzoll-0.1-SNAPSHOT.jar server config.yml`

## Connect with the frontend

- open `html-frontend\chzoll.html` in your a widely used browser (with JavaScript enabled)
